# Installation Lot 2

Copie ces fichiers dans ton Lot 1, en remplaçant ceux qui existent.

Ajoute le contenu de `docs/CSS_AJOUT_LOT2.css` à la fin de `app/globals.css`.

Ajoute dans Vercel :

`SUPABASE_AUDIO_BUCKET=equilibria-audios`

Dans Supabase Storage, crée un bucket privé :

`equilibria-audios`

Test :
1. Connecte-toi.
2. Va sur `/admin/upload`.
3. Uploade un M4A ou MP3 pour le jour 1.
4. Va sur `/app`.
5. Ouvre le chapitre 1.
6. Clique “Charger l’audio sécurisé”.
